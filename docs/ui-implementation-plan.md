# 🎨 AI 반려로봇 UI 화면 설계 작업 계획서

> **프로젝트**: AI 반려로봇 보호자 웹앱 (프로젝트명 미정)  
> **디자인 컨셉**: 토스(Toss) 스타일 미니멀 UI  
> **플랫폼**: 반응형 웹 (모바일 우선)  
> **CSS 프레임워크**: Tailwind CSS v3  
> **작성일**: 2026-01-20

## 1. 디자인 시스템 (Design Tokens)

### 1.1 컬러 팔레트

```
┌─────────────────────────────────────────────────────────────┐
│  PRIMARY (토스 블루 계열)                                    │
```

### 1.2 타이포그래피

| 용도 | 크기 | Weight | 행간 | 사용처 |
|------|------|--------|------|--------|
| **Display** | 32px | Bold (700) | 1.2 | 메인 대시보드 인사말 |
| **Heading 1** | 24px | SemiBold (600) | 1.3 | 화면 제목 |
| **Heading 2** | 20px | SemiBold (600) | 1.4 | 섹션 제목 |
| **Heading 3** | 16px | Medium (500) | 1.4 | 카드 제목 |
| **Body** | 14px | Regular (400) | 1.6 | 본문 텍스트 |
| **Caption** | 12px | Regular (400) | 1.5 | 보조 설명 |
| **Large Number** | 48px | Bold (700) | 1.0 | 숫자 강조 (배터리 등) |

**권장 폰트**: `Pretendard` (한글 최적화) 또는 `Inter` (영문)

### 1.3 간격 시스템 (4px 기준)

```
┌─────────────────────────────────────────────┐
│  space-1   4px    아이콘 내부 간격           │
│  space-2   8px    인라인 요소 간격           │
│  space-3   12px   작은 패딩                 │
│  space-4   16px   기본 패딩                 │
│  space-5   20px   카드 내부 패딩             │
│  space-6   24px   섹션 간격                 │
│  space-8   32px   큰 섹션 간격              │
│  space-10  40px   화면 여백                 │
└─────────────────────────────────────────────┘
```

### 1.4 Border Radius

| 이름 | 값 | 용도 |
|------|-----|------|
| `rounded-sm` | 8px | 작은 버튼, 입력 필드 |
| `rounded-md` | 12px | 태그, 뱃지 |
| `rounded-lg` | 16px | 카드, 모달 |
| `rounded-xl` | 20px | 대형 카드 |
| `rounded-full` | 9999px | 아바타, 원형 버튼 |

### 1.5 그림자 (Elevation)

```css
shadow-sm:   0 1px 2px rgba(0,0,0,0.05)      /* 카드 기본 */
shadow-md:   0 4px 6px rgba(0,0,0,0.07)      /* 호버 상태 */
shadow-lg:   0 10px 15px rgba(0,0,0,0.1)     /* 모달, 드롭다운 */
shadow-danger: 0 0 0 4px rgba(240,68,82,0.2) /* 긴급 상황 카드 */
```

## 2. 화면 구조 및 라우팅

### 2.1 전체 화면 목록

| # | 화면명 | 경로 | 설명 | 접근성 |
|---|--------|------|------|--------|
| 1 | 로그인 | `/login` | 이메일/비밀번호 로그인 | 공개 |
| 2 | 회원가입 | `/signup` | 신규 가입 | 공개 |
| 3 | 노인 선택 | `/elders` | 담당 노인 목록 (복지사) | 인증 필요 |
| 4 | 홈 (대시보드) | `/elders/:id` | 실시간 상태 요약 | 인증 필요 |

| 6 | 로봇 제어 | `/elders/:id/robot` | 로봇 상태 & 원격 제어 | 인증 필요 |
| 7 | 기록 | `/elders/:id/history` | AI 리포트 & 활동 로그 | 인증 필요 |
| 8 | 약 관리 | `/elders/:id/medications` | 복용 현황 & 약 정보 | 인증 필요 |
| 9 | 알림 | `/notifications` | 전체 알림 히스토리 | 인증 필요 |
| 10 | 설정 | `/settings` | 프로필 & 앱 설정 | 인증 필요 |
| 11 | 긴급 상황 | `/emergency/:id` | 풀스크린 경고 화면 | 인증 필요 |
| 12 | 로봇 LCD 미러링 | `/elders/:id/robot/lcd` | 로봇 화면 미리보기 | 인증 필요 |

### 2.2 네비게이션 구조

> **토스 스타일**: 알림은 하단 탭이 아닌 **상단 헤더의 🔔 아이콘**으로 접근

```
┌─────────────────────────────────────────────────────────────────┐
│  [김옥분 ▼]                                       🔔 [3]        │  ← 알림 벨 아이콘
│  ┌──────────────────────────────────────────────────────────┐  │
│  │                     CONTENT AREA                         │  │
│  │                                                          │  │
│  │                                                          │  │
│  │                                                          │  │
│  │                                                          │  │
│  └──────────────────────────────────────────────────────────┘  │
│                                                                 │
│  ┌──────┬──────┬──────┬──────┬──────┐                        │
│  │  🏠  │  🤖  │  💊  │  📅  │  ⚙️  │   ← 하단 탭 바 (5개)    │
│  │  홈  │ 로봇 │  약  │ 일정 │ 더보기│                         │
│  └──────┴──────┴──────┴──────┴──────┘                        │
└─────────────────────────────────────────────────────────────────┘

※ 탭 순서: 홈 → 로봇(상태확인) → 약(매일확인) → 일정 → 더보기(기록, 설정)
※ 알림은 상단 🔔 아이콘 클릭 시 알림 화면으로 이동
```

### 2.3 반응형 브레이크포인트

| 이름 | 너비 | 레이아웃 |
|------|------|----------|
| **Mobile** | < 640px | 단일 컬럼, 하단 탭 바 |
| **Tablet** | 640px ~ 1024px | 2컬럼 그리드 |
| **Desktop** | > 1024px | 사이드바 + 메인 콘텐츠 |

## 4. 로봇 LCD 화면 설계 (라즈베리파이 키오스크)

> **커뮤니케이션 방식**
> - **음성(TTS)** = 주(主): 로봇이 음성으로 먼저 말함
> - **LCD 텍스트** = 보조: 음성을 놓쳤을 때 시각적으로 확인

> **디자인 원칙**: 큰 글씨, 단순한 UI, 터치 친화적, 표정 이모티콘 강조


### 4.1 LCD 화면 목록

| # | 화면명 | 트리거 | 설명 |
|---|--------|--------|------|
| 1 | 대기/기본 화면 | 평상시 | 표정 + 시간 + 다음 일정 |
| 2 | 인사 화면 | 기상/귀가 감지 | 아침/귀가 인사 |
| 3 | 복약 알림 | 약 시간 도래 | 약 복용 확인 버튼 |
| 4 | 일정 알림 | 일정 N분 전 | 일정 안내 |
| 5 | 대화/듣는 중 | 음성 인식 활성화 | 파동 애니메이션 |
| 6 | 긴급/낙상 감지 | 낙상 감지 시 | SOS 버튼 |
| 7 | 충전 중 | 도킹 스테이션 | 충전 상태 표시 |

### 4.9 LCD 디자인 가이드라인

| 항목 | 값 | 이유 |
|------|-----|------|
| **폰트 크기** | 최소 24px, 제목 40px+ | 고령자 가독성 |
| **버튼 크기** | 최소 높이 80px | 터치 정확도 |
| **색상 대비** | WCAG AAA 준수 | 시인성 |
| **표정 아이콘** | 128x128px 이상 | 감정 전달 |
| **애니메이션** | 느리고 부드럽게 (0.5s+) | 인지 시간 확보 |
| **야간 모드** | 21:00 이후 어두운 테마 | 수면 방해 최소화 |

### 4.1 컴포넌트 목록

| 컴포넌트 | 용도 | 변형 |

| `Badge` | 상태 표시 | success, warning, danger, info |
| `Avatar` | 프로필 이미지 | sm, md, lg |
| `BottomTabBar` | 하단 네비게이션 | 5개 탭 고정 |
| `StatusIndicator` | 연결 상태 | online, offline, warning |
| `Timeline` | 활동 로그 | - |
| `Calendar` | 일정 캘린더 | 월간, 주간 |
| `Toast` | 알림 토스트 | success, error, info |
| `Modal` | 모달창 | center, bottom-sheet |
| `Skeleton` | 로딩 상태 | - |
